
 ATM Machine – Manual Testing Project

  Project Overview

This project simulates ATM machine functionality and demonstrates manual testing techniques.
It includes test case design, positive & negative testing, defect reporting, and validation of banking transactions.

The goal of this project is to validate ATM system behavior under different user scenarios.

---

Objectives

* Validate ATM core functionalities
* Perform positive and negative testing
* Identify defects and document bug reports
* Verify system behavior after defect fixes

---

 Modules Tested

* Card Insertion
* PIN Authentication
* Balance Inquiry
* Cash Withdrawal
* Mini Statement
* Transaction History
* Card Ejection

---

Test Case Documentation

Test cases are designed with:

* Test Case ID
